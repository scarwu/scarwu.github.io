用 Raspberry Pi 玩 PS 遊戲

2013-05-29  /  IT  /  Linux Raspberry Pi

前言

Raspberry Pi小巧好用拿來打電動很剛好，當作PS遊戲模擬器也是游刃有餘。

所以在這邊紀錄如何在Raspberry Pi上安裝PCSX ReARMed，它是我們所熟知的PS模擬器PCSX，但它是針對ARM CPU的特製版。

安裝過程

下面的範例是直接在RPi上編譯，過程中沒有用到Cross Compile所以編譯時間會比較久。

安裝編譯環境

sudo apt-get uptdate && sudo apt-get upgrade
sudo apt-get install git build-essential libpng-dev libx11-dev autoconf

編譯SDL

wget http://www.libsdl.org/release/SDL-1.2.15.tar.gz
tar zxvf SDL-1.2.15.tar.gz
cd SDL-1.2.15
./autogen.sh
./configure && make
sudo make install

編譯PCSX ReARMed

git clone git://github.com/notaz/pcsx_rearmed.git
cd pcsx_rearmed
git submodule init && git submodule update
./configure

編輯makefile，並將-I/opt/vc/include/interface/vmcs_host/linux 加入CFLAGS。

# add -I/opt/vc/include/interface/vmcs_host/linux to CFLAGS
vim Makefile
make

啟動

在啟動前記得準備好搖桿以及遊戲映像檔。

# Start PCSX
./pcsx
