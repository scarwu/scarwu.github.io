Android 開發環境設置

2011-09-26  /  IT  /  Android Note

前言

最近因為課程需要所以要架設Android開發的環境給學生使用，無奈學校的機器只有Windows，當然我自己還是使用Linux，所以就在這邊紀錄一下如何在Windows下架設環境。

下載

  • JDK 7 - jdk-7-windows-i586.exe
  • Android SDK - android-sdk_r15-windows.zip
  • ADT 15.0.0 - ADT-15.0.0.zip
  • Eclipse Classic 3.7 - Windows 32 Bit

安裝步驟

  1. 安裝 JDK
  2. 在 C:\ 底下新增資料夾 "Android"
  3. 將ADT, Andoird SDK, Eclipse放入 C:\Android\
  4. 將 Eclipse, Adroid SDK 解壓縮出來，並修改為 C:\Android\eclipse, C:\Android\android-sdk-windows
  5. 開啟 C:\Android\android-sdk-windows\SDK Manager.exe 並更新至最新版
  6. 接著開啟 C:\Android\eclipse\eclipse.exe 安裝 ADT
  7. 設定 [Help] > [Install New Software] > [Add] > [Archive] 選擇 C:\Android\ADT-12.0.0.zip
  8. 接著選擇並安裝 Android DDMS, Android Development Tools 並重新啟動 Eclipse
  9. 設定 [Window] > [Preferences] > [Android] > [Browse] 選擇 C:\Android\android-sdk-windows 接著按下 [Apply] > [OK]
  10. 完成

Update: 2011/10/31 - 因為 Android 4.0 已經出了所以這邊做了點修正，不過安裝方式還是差不多。
