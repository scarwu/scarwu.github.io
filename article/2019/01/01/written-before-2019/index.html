寫在 2019 開頭

2019-01-01  /  Life  /  Feeling

在前年 2017 曾經對於 2018 有所期許，沒想到在去年 2018 這一年間的轉變真的是非常巨大出乎意料之外。

我放棄了原有的舒適圈還是踏出了一小步，原本我希望這會是一大步但計劃趕不上變化，雖然失敗了但並沒有後悔。

Side Projects

因為有了一些閒暇的時間所以個人專案有了滿大的進步。

魔物獵人世界計算機

這個計算機是個意外產物，起初我只是因為想要配個裝但是工具不夠用，不過最近開始了新生活沒什麼時間，所以沒有更新裝備列表這個要找時間補上。

Oni

配合 Pointless 的改良，在背後的核心也是勢必要進化，所以將原本的核心的 NanoCLI 整了進來，再把原本的 Web 架構做了些調整，接下來會對於 CLI 進行強化。

Pointless & Theme

對比 2017 年的期許，這個工具真的改變許多，我將原本的底層從 NanoCLI 抽換成 Oni，並重構符合 PSR 標準，還將原本的 Built-in Server 改由 Oni 負責直接動態產出頁面。

並且將原本的 Theme 切出專案之外，並且可以配合新的 Built-in Server 開發，且使用現代化的前端工具，新的部落格樣式就是這樣開發出來的。

攝影及重車

在這一年間去了許多地方也拍了許多相片，雖然技術可能還是很粗淺，但可以到全台各地並且留下紀念，將兩個興趣結合也是不錯的事。
